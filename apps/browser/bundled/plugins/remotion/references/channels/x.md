## Video formats

Limits: aspect `1:2.39` to `2.39:1`. Min `32×32`. Max `1920×1200` landscape / `1200×1900` portrait. Max `40 fps`. Max `25 Mbps`.

### Portrait

- Res: `720×1280`
- Aspect: `9:16`
- Safe area: no official spec. Keep inner margins for UI

### Landscape

- Res: `1280×720` safe, or `1920×1080`
- Aspect: `16:9`

### Square

- Res: `720×720` or `1080×1080`
- Aspect: `1:1`

## Encoding

- Container: `MP4` best, or `MOV`
- Video codec: `H.264`
- Audio codec: `AAC`
- FPS: `30`
- Bitrate: `<25 Mbps`

## Limits

- File: `512 MB` non-premium | `8-16 GB` premium/pro, docs conflict
- Formats: `MP4`, `MOV`
- Length: `140s` non-premium | `3-4h` premium
- Auto-loop: videos `<=60s`
- Timestamps: up to `50`

## Style

- Autoplay muted. Optimize sound-off
- Use `.srt` or burnt-in captions
- Hook in first `3s`
- Aim `<2 min` for completion
- Upload native. External links weaker, no autoplay

## Post copy

- Limit: `280` chars standard | `25,000` premium
- Links count as `23` chars
- Keep copy minimal. Better brand recall
- Use hashtags strategically
- Pro Media / Media Studio: separate title `70` chars, description `200` chars

## Thumbnails

- Custom thumb: premium via Media Studio / Pro Media
- A/B test: not natively verified
- First frame matters. Often preview before autoplay

## Engagement

- Watch: views, retention/completion, watch time
- Viral signals: reposts, replies, bookmarks
- Conversion: link clicks, profile visits
- Downloads: creators can allow premium-user downloads
