Layout definition must include:

- Which elements are used. Define following:
  - Type of element (text, video, demo component, etc.)
  - Size, Variant, Color
  - Enter/Exit Animations/Transitions
  - Moving during sequence
  - Optional or mandatory
  - Position (absolute/relative, distance or border)
  - Resizing & Truncation behavior
- Relative position of elements to each other.
  - What is center point of the scene. How does the scene move.
- Transition to other layouts
  - MMultiple variants for enter and exit possible. Define when to use which variant.
