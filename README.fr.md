## À propos du projet

**stagewise** est un IDE agentique open source pour les développeurs, avec un agent de programmation intégré.

- **Naviguer et construire** dans le même outil — sans changement de contexte
- **Travailler avec un agent de programmation** qui a **un accès complet à la console et au débogueur de votre onglet**
- **Effectuer des modifications de test temporaires** ou **connecter une base de code** pour des éditions permanentes
- **Rétro-ingénierie** des composants, des systèmes de styles et des palettes de couleurs de n'importe quel site web
- **Intégration IDE** pour visualiser et appliquer les modifications de code dans votre éditeur préféré
- **Apportez votre propre clé API (BYOK)** — prise en charge pour tous les fournisseurs d'IA

## Démarrage

Téléchargez stagewise sur [stagewise.io](https://stagewise.io) et suivez le court guide d'intégration pour configurer votre compte.
