## 프로젝트 소개

**stagewise**는 코딩 에이전트가 내장된 개발자 전용 브라우저입니다.

- **같은 도구에서 탐색과 개발** — 컨텍스트 전환 없음
- **코딩 에이전트와 함께 작업** — 탭의 **콘솔과 디버거에 완전한 접근 권한** 보유
- **임시 테스트 변경** 또는 **코드베이스 연결**로 영구 편집
- 모든 웹사이트의 컴포넌트, 스타일 시스템, 색상 팔레트를 **리버스 엔지니어링**
- **IDE 연동** — 선호하는 에디터에서 코드 변경 사항 확인 및 적용
- **API 키 직접 사용 (BYOK)** — 모든 AI 공급자 완전 지원

## 시작하기

[stagewise.io](https://stagewise.io)에서 stagewise를 다운로드하고 짧은 온보딩 가이드를 따라 계정을 설정하세요.

## 코딩 구독 활용하기

모든 주요 모델 공급자에 직접 API 키를 사용할 수 있습니다(BYOK). 완전히 커스텀한 공급자(로컬 추론 포함!)와 커스텀 모델도 등록할 수 있습니다.
